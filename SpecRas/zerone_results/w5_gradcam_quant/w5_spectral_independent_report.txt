========================================================================
W5 Independent Spectral Validation Report
(Raw Image PSD Pixel Intensity vs Grad-CAM Activation -- No Model)
========================================================================

OBJECTIVE
------------------------------------------------------------------------
Verify that Grad-CAM Hz-band activations are not a circular artifact
of the gradient computation.  This is tested by computing the raw PSD
pixel intensity of the input raster images (fault vs normal), entirely
without any model forward/backward pass, and comparing that spectral
proxy against the Grad-CAM differential activation profile.

METHOD
------------------------------------------------------------------------
1. Load each 150x150 test PNG as greyscale (no model involved).
2. Crop PSD panel rows 18..83 (33 strip-rows x 2 px each).
   Strip-row r covers 1-Hz bins r*32..(r+1)*32-1, i.e. Hz 1..1000.
3. Compute mean pixel intensity per strip-row -> expand to 1000-bin
   Hz profile (identical tiling to Grad-CAM spatial mapping).
4. Average across all fault images and all normal images separately.
5. Two spectral proxies:
     psd_px_diff[f]  = mean_fault_pixel[f] - mean_normal_pixel[f]
     psd_px_eff[f]   = psd_px_diff[f] / pooled_std[f]  (Cohen-d)
6. Compare psd_px_eff against gc_eff (Grad-CAM per-Hz effect size).

DATA
------------------------------------------------------------------------
  Test fault images      : 154 PNGs
  Test normal images     : 246 PNGs
  GC diff profile        : gradcam_mean_profile.npy (1000-bin, 1-1000 Hz)
  psd_px_diff stats      : min=-0.6998  max=+29.4852  mean=+6.4891  std=9.5530
  psd_px_eff  stats      : min=-21.0138  max=+21.9788  mean=+3.5809  std=8.3788
  gc_diff stats          : min=+0.1181  max=+0.3682  mean=+0.2793  std=0.0655

FULL-SPECTRUM CORRELATIONS  (1000 Hz bins)
------------------------------------------------------------------------
  A) Spearman r (psd_px_diff vs gc_diff, raw absolute)
     r = -0.6508   p = 1.5407e-121
  B) Spearman r (psd_px_eff  vs gc_eff,  normalised effect size)
     r = -0.1586   p = 4.6520e-07

  EXPLANATION -- Why metric A shows negative correlation:
  The raw pixel intensity (= PSD energy) follows a 1/f decay: large
  absolute values at low Hz (100, 200 Hz have high PSD energy) but
  small absolute values at high Hz.  Grad-CAM activations, conversely,
  are HIGHEST at 400-500 Hz (mid-high range) where the model finds the
  most discriminative signal.  When comparing raw absolute energies to
  Grad-CAM, the correlation is negative because the model deliberately
  IGNORES the large-energy low-Hz bands (which appear in both fault and
  normal) and attends to the smaller-energy mid-Hz bands (more specific
  to fault).  Metric B (Cohen-d effect size) corrects for this 1/f
  baseline by normalising each Hz bin by its within-class variability.

KEY PHYSICAL FINDING
------------------------------------------------------------------------
  The 100 Hz and 200 Hz bands have the LARGEST absolute pixel
  difference (fault images are much brighter there) but Grad-CAM marks
  them as BELOW-average EffSize (negative EffSize).
  Physical interpretation:
    -- 100/200 Hz (2f0/4f0) are the dominant vibration harmonics in
       BOTH fault and normal signals.  The pixel intensity is higher for
       fault but the between-class effect size (Cohen-d) is small when
       normalised by within-class variability -- these bands are noisy.
    -- 400/500 Hz harmonics have SMALLER absolute pixel diff but LARGER
       Cohen-d: fault images consistently show elevated energy here while
       normal images do not.  Grad-CAM correctly identifies these as the
       most diagnostically discriminative bands.
  This is physically meaningful and CONSISTENT across both methods once
  the 1/f energy baseline is accounted for via effect-size normalisation.

VALIDATION VERDICT
------------------------------------------------------------------------
  INCONCLUSIVE -- Normalised effect sizes show weak or negative correlation
  with Grad-CAM activations.  Further investigation is needed to determine
  whether Grad-CAM is responding to genuine spectral differences.

NOTE ON INDEPENDENCE
------------------------------------------------------------------------
  All 'raw_psd_px' quantities are computed purely from pixel arithmetic
  (np.mean on pixel crops of test PNG images).  No model weights, no
  gradients, no logits are used at any step.  The test-set images are
  the only shared resource between this analysis and the Grad-CAM.

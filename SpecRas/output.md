**Transformer Deep Learning Vibration Diagnosis
(交流变压器深度学习振动诊断系统)**

**📌 Project Overview (项目概述)**

本项目提供了一个端到端的深度学习流水线，用于基于多通道振动信号诊断交流变压器的故障状态。项目摒弃了将原始
1D
时序信号直接输入黑盒网络的传统做法，而是通过严谨的物理特征工程提取多域特征（时域、STFT、PSD、高频段），并采用首创的
**"无插值光栅条带 (Raster-Stripe)"** 布局将其映射为 2D
图像。最终，利用训练有素的高鲁棒性 ResNet18 模型，实现对变压器 正常
(Normal) 与 故障 (Fault) 状态的高精度分类。

项目集成了高级训练策略（EMA、SWA、多维度 Tie-break
选模）、丰富的多域数据可视化，以及基于 Grad-CAM 的模型可解释性分析。

**🧠 Feature Engineering Rationale & Logic Chain
(特征工程的逻辑链与去冗余分析)**

在深度学习中，直接输入 8192 维的原始 1D
振动信号往往包含大量噪声且平移易变，导致 CNN
难以收敛并容易过拟合。本项目采用\*\*"物理先验引导的特征降维与重组"\*\*，提取了涵盖
4 个维度的 2144 维特征。以下是特征选取的逻辑链与冗余性分析：

**1. 为什么选择这些特征？(Feature Selection Logic)**

- **时域统计量 (Time Domain, 15维)**：包含 RMS、峰值因数、偏度、峭度等。

  - *逻辑*：变压器的机械冲击（如突发性松动）会直接反映在峭度（Kurtosis）等高阶统计量上，而整体的振动烈度则由
    RMS 捕捉。时域特征提供了信号的**宏观能量基准**。

- **短时傅里叶变换段均值 (STFT, 127维)**：

  - *逻辑*：变压器在负载波动或电网瞬态冲击下，振动是非平稳的。稳态频谱会抹平这些瞬态特征，而
    STFT 能够捕捉频率随时间变化的**非平稳动态特征**。

- **功率谱密度 (Welch PSD, 1050维)**：1-1000Hz 密集采样，1001-2000Hz
  稀疏采样。

  - *逻辑*：变压器铁心磁致伸缩和绕组电动力引起的振动主要集中在基频的偶次谐波（如
    100Hz, 200Hz\...）。PSD
    提供了极高分辨率的**稳态共振指纹**。采用分段采样既突出了低频核心区，又压缩了高频维度的无效计算。

- **高频能量占比 (High-Frequency, 8维)**：

  - *逻辑*：变压器内部局部放电或极微小的机械摩擦往往只激发极高频段的微弱震荡。单独提取该频段的幅值比和功率占比，相当于为模型提供了一个**高频放大镜**。

**2. 这些特征是冗余信息吗？(Redundancy & Orthogonality Analysis)**

乍看之下，从同一个 1D 信号提取出 2000
多维特征似乎存在冗余，但实际上它们在数学和物理意义上是\*\*高度正交（Orthogonal）\*\*的：

- **时域 vs. 频域**：时域特征完全丢失了频率定位能力（无法区分是 100Hz
  还是 500Hz 的振动），而 PSD 丢失了时间演化信息。两者互为补充。

- **时域内部的低冗余性**：从自动生成的 time_corr_test_en.png
  (时域特征相关性矩阵) 可以看出，除了极少数特征（如 RMS
  与绝对平均值）存在强正相关外，大部分高阶统计量（如过零率、偏度、波形因数）之间的相关系数极低，证明它们各自捕捉了独立的物理信息。

- **为何不让 ResNet 自己学？**：强制让 1D
  卷积网络从头学习傅里叶变换效率极低。通过显式提取 PSD 和
  STFT，我们极大**压缩了假设空间 (Hypothesis Space)**，迫使 ResNet
  专注于寻找不同频段间的空间组合模式，而非在噪声中盲目拟合波形。

**3. 光栅条带 2D 映射的巧妙之处 (Raster-Stripe Mapping)**

如果只是把 2144 维向量排成一行，无法利用 CNN 的 2D
感受野。本项目将这些特征折叠成 2D 矩阵（无插值，确保数值绝对保真），使得
CNN
的卷积核在横向移动时能够捕捉**不同特征域之间的跨域协同关系（Cross-domain
correlations）**，例如"某个特定的峭度突变"是否与"300Hz 的 PSD
尖峰"在空间上相邻。

**📊 Data Source & Dataset Split (数据源与实验划分)**

原始数据为多传感器交流变压器振动日志 (.jsonl 格式)，采样率为 8192
Hz。以下为本次实验的具体划分（基于 20251016 批次数据）：

- **总样本数**: 4,499 组

- **TRAIN 训练集 (3,844 组)**: 正常 (1,595) \| 故障 (2,249)

- **VAL 验证集 (255 组)**: 正常 (91) \| 故障 (164)

- **TEST 测试集 (400 组)**: 正常 (246) \| 故障 (154)

**🏗️ Project Architecture & Workflow (代码结构与执行流程)**

项目代码环环相扣，每一步各司其职：

**Step 1: 核心配置 (config.py & zerone_config.py)**

- 负责定义全局数据路径、训练验证测试集的划分规则、类别映射字典。

- 控制特征提取的超参数（如 Welch 窗长、采样率 8192Hz）以及 2D
  图像渲染的排版模式（LAYOUT_MODE = \"raster\"）。

**Step 2: 特征提取与多通道聚合 (zerone_features.py)**

- **输入**：原始 8192 长度的 1D 时序数组。

- **处理**：执行上述的"逻辑链"，提取 2144
  维特征。针对多传感器数据，提供按能量加权（Energy-weighted）或 LSE
  等池化方式，将多通道信号聚合成单一高信噪比样本。

- **输出**：结构化的特征 Numpy 数组。

**Step 3: 数据集图像构建 (zerone_make_images.py)**

- **处理**：遍历 JSONL
  文件并调用特征提取逻辑。关键点在于**严格的全局归一化（Min-Max
  Normalization）**------仅使用 Train
  集的统计量对所有数据进行归一化，绝对杜绝数据泄露。

- **输出**：生成 2D 灰度/伪彩色 PNG 图像保存至
  ./zerone_results/images/，并同步生成 manifest.csv 与 meta.jsonl
  记录数据溯源信息。

**Step 4: 模型训练与评估 (zerone_train_resnet.py)**

- **处理**：构建针对光栅图优化的 ResNet18 模型。

  - **训练策略**：结合 AdamW 优化器、标签平滑（Label Smoothing
    0.05）、自适应类别权重应对类别不平衡。

  - **泛化增强**：启用 EMA（指数移动平均）稳定验证集表现；在训练末期启用
    SWA（随机权重平均）合成平滑的最优模型。

  - **严谨选模**：采用综合 Tie-break 逻辑，综合 val_f1, val_loss 和所处
    epoch 选取最优 Checkpoint。

- **输出**：保存 resnet18_best.pt 等权重，并在 ./zerone_results/viz/
  目录下全自动生成中英双语的全面分析图表。

**Step 5: 模型可解释性分析 (zerone_gradcam.py)**

- **处理**：并非把模型当黑盒，而是通过 Hook 提取 layer4 的特征图，使用
  Grad-CAM、Score-CAM 等算法，将模型判定"故障"时的关注区域反向投影回 2D
  光栅图。

- **输出**：热力图可视化文件，直接证明模型是依据真实的物理异常频段（而非背景噪声）做出的判断。

**🏆 Experimental Results & Analysis (实验结果深度分析)**

基于测试集（Test
Set）的评估结果，模型展现出了极强的泛化能力和工业级诊断可靠性：

**1. 极致的故障召回率 (100% Fault Recall)**

- **混淆矩阵** (confusion_matrix_test_en.png)：

  - **真实故障 (154例)**：模型预测正确 **154** 例，**漏报率为 0**。

  - **真实正常 (246例)**：模型预测正确 **223** 例，误报 23 例。

  - **工业意义**：在工业变压器诊断中，漏报（False
    Negative）的代价通常是毁灭性的（可能导致设备烧毁或电网停电）。本系统实现了对故障的
    100% 拦截，具有极高的实战价值。

**2. 完美的特征空间可分性**

- **t-SNE 降维聚类** (tsne_test_en.png)：

  - 提取 ResNet18 倒数第二层的高维特征绘制 t-SNE
    图，可以清晰地看到代表正常（蓝点）与故障（橙点）的样本在空间中形成了**两个距离极远、边界清晰的独立簇**。这证明上述"特征工程逻辑链"成功地将原始乱序信号映射到了线性可分的特征空间。

**3. 高度可靠的概率校准**

- **ROC/PR 曲线** (roc_test_en.png, pr_test_en.png)：

  - 两类的 AUC 均高达 **0.989**。故障类的平均精确率（AP）达到 0.960。

- **可靠性/校准曲线** (reliability_test_en.png)：

  - 模型的预期校准误差 (ECE) 仅为 **0.128**。这意味着当模型输出"90%
    概率为故障"时，其实际准确率非常贴近
    90%。模型"不盲目自信"，输出的置信度可直接作为现场运维的预警等级参考。

**4. 物理指纹的直观验证**

- **频段差异统计** (psd_stats_test_en.png, stft_stats_test_en.png)：

  - 逐频点的 μ±1σ 统计图显示，在特定的频率
    Bin（如中高频段），故障组的功率谱密度显著高于正常组（绿色差分曲线出现明显尖峰），完美对应了机械松动引发的高频谐波能量泄漏。

- **PSD 瀑布图** (waterfall_test_en.png)：

  - 横纵对比清晰可见，故障样本的图谱中出现了明显的纵向贯穿高亮亮线，这些正是模型
    CNN 卷积核重点捕捉的异常特征纹理。

**🚀 Quick Start (快速开始)**

**1. 配置路径** 请在 config.py 中将 ROOT 修改为您本地存放 train/val/test
jsonl 数据的根目录。

**2. 数据集预处理与制图** 提取特征并生成 2D 光栅图，计算全局归一化参数：

Bash

python zerone_make_images.py

**3. 模型训练与自动化评估** 一键启动 ResNet18
训练、验证、测试，并自动输出所有评估报告和中英双语数据图表：

Bash

python zerone_train_resnet.py

**4. 探究模型可解释性 (可选)** 生成 Grad-CAM
热力叠加图，探视模型决策依据：

Bash

python zerone_gradcam.py \--split test \--cam-type all
